## Executar o projeto

<br>
<li>
Faça um clone para o diretório desejado
</li>
<li>Na pasta api e web digite o comando abaixo para instalar as dependências:
</li>
<br>

### `yarn install`

<br>
<li>
Para rodar a aplicação:
</li>
<br>

### `yarn start`
